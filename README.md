# Simple Interest Calculator

This project calculates simple interest based on user input for principal, rate, and time.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Example
If Principal = 1000, Rate = 5%, Time = 2 years  
Then Simple Interest = (1000 × 5 × 2) / 100 = 100
